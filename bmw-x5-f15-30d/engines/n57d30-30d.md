# N57D30 / BMW X5 F15 xDrive30d

## Короткий вердикт

N57D30 в BMW X5 F15 xDrive30d — главный и единственный мотор, который имеет смысл рассматривать для нашей задачи.

Это рядная дизельная шестерка BMW: тяговитая, понятная профильным сервисам, достаточно надежная при нормальном обслуживании и хорошо подходящая тяжелому X5.

Главная идея:

Ищем X5 F15 xDrive30d без чипа, без криво удаленной экологии, с понятной историей обслуживания, живой топливной системой, обслуженной АКПП, обслуженной раздаткой и одинаковой резиной.

- Модель: BMW X5 F15 xDrive30d

- Объем: 3.0 л

## Что проверить по мотору

Обязательно:

- холодный запуск;
- посторонние звуки на холодную;
- шум цепей;
- коррекции форсунок;
- давление топлива;
- состояние ТНВД;
- состояние турбины;
- EGR;
- DPF;
- впуск;
- сажу и отложения во впуске;
- течи масла;
- течи антифриза;
- температуру двигателя;
- ошибки по всем блокам;
- дымность под нагрузкой.

## Топливная система

На пробегах 120–200 тыс. км топливную систему нужно проверять обязательно.

Смотреть:

- коррекции форсунок;
- давление в рампе;
- ошибки по давлению топлива;
- работу ТНВД;
- металлическую стружку, если есть подозрения на ТНВД;
- запуск на холодную;
- запуск на горячую;
- ровность работы мотора.

Плохой холодный запуск, ошибки по давлению топлива, неровная работа и плохие коррекции форсунок — повод не покупать без глубокой диагностики.

## EGR / DPF / впуск

Желательно искать машину с живой экологией.

Проверять:

- состояние EGR;
- состояние DPF;
- частоту регенераций;
- ошибки по сажевому фильтру;
- давление до/после DPF;
- состояние впуска;
- следы удаления экологии;
- следы вмешательства в прошивку.

Удаленная экология сама по себе не всегда приговор, но для нашей покупки это красный флаг, особенно если нет документов и понятно выполненной работы.

## Цепи ГРМ

N57 считается крепким мотором, но цепи не вечные.

Проверять:

- шум на холодном запуске;
- металлический шелест;
- ошибки по фазам;
- историю работ по ГРМ;
- косвенные признаки большого пробега.

Если есть явный шум цепи — это повод закладывать дорогой ремонт или отказаться от машины.

## Турбина

Проверять:

- люфт;
- свист;
- масло во впуске;
- недодув;
- передув;
- ошибки по наддуву;
- тягу под нагрузкой;
- дымность.

Турбина на возрастной машине может быть живой, но ее состояние обязательно нужно проверять диагностикой и тест-драйвом.

## Что не брать

Не наш вариант:

- чип-тюнинг;
- Stage 1 / Stage 2 без документов;
- удаленный DPF / EGR без понятной истории;
- дымящий мотор;
- плохой холодный запуск;
- ошибки по топливной;
- ошибки по наддуву;
- грубая работа мотора;
- сильные течи;
- уходящий антифриз;
- свежевымытый мотор перед продажей;
- продавец не готов к профильной диагностике.

## Связанные узлы

Для X5 F15 мотор нельзя проверять отдельно от трансмиссии.

Обязательно смотреть вместе с мотором:

- АКПП ZF 8HP;
- раздатку;
- редукторы;
- кардан;
- приводы;
- xDrive;
- одинаковость резины;
- вибрации под нагрузкой;
- рывки при разгоне.

Даже живой мотор не спасает машину, если раздатка и xDrive уже уставшие.
