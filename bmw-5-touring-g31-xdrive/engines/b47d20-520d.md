# B47D20 / BMW 520d xDrive Touring G31

## Короткий вердикт

B47D20 в BMW 520d xDrive Touring G31 — допустимый резервный вариант, но не главный.

Это рациональный 2.0 дизель: экономичный, массовый, понятный в обслуживании. Но для большой 5 серии Touring он дает скорее достаточную, чем приятную динамику.

Главная идея:

520d xDrive можно рассматривать только при очень хорошем состоянии, прозрачной истории, адекватной цене и после тест-драйва. Покупать 520d с мыслью “потом прошью” не надо.

## Базовые данные

- Модель: BMW 520d xDrive Touring G31
- Годы для поиска: 2017–2020
- Двигатель: B47D20
- Тип: рядный 4-цилиндровый дизель
- Объем: 2.0 л
- Мощность: 190 л.с.
- Крутящий момент: около 400 Нм
- Топливная система: Common Rail
- Наддув: турбина
- ГРМ: цепной
- Коробка: ZF 8HP
- Привод: xDrive

## Почему его можно рассматривать

Плюсы:

- ниже расход;
- ниже налог;
- проще и дешевле обслуживание, чем у 530d;
- мотор массовый;
- больше выбор машин;
- легче найти спокойную комплектацию;
- хороший вариант, если динамика устраивает.

Минусы:

- динамика скромнее;
- меньше запаса на трассе;
- мотор работает с большей нагрузкой;
- меньше эмоций;
- при полной загрузке может ощущаться слабоватым;
- чип-тюнинг сильно портит смысл покупки;
- дизельная экология все равно остается.

## Когда 520d можно брать

520d xDrive можно рассматривать, если:

- состояние машины очень хорошее;
- есть прозрачная история обслуживания;
- пробег подтвержден;
- машина без чипа;
- экология живая;
- АКПП работает идеально;
- xDrive без ошибок и рывков;
- цена заметно лучше, чем у 530d;
- после тест-драйва динамика устраивает.

## Когда 520d брать не надо

Не надо брать, если:

- хочется динамики;
- планируется чип;
- машина тяжелая по опциям и на больших колесах;
- есть ошибки по экологии;
- нет истории обслуживания;
- мотор работает грубо;
- есть дымность;
- есть пинки АКПП;
- есть рывки xDrive;
- цена близка к живым 530d.

Если цена почти как у 530d, лучше искать 530d.

## Что проверить по мотору

Обязательно:

- холодный запуск;
- шум цепи;
- ровность работы;
- коррекции форсунок;
- давление топлива;
- ТНВД;
- турбину;
- EGR;
- DPF;
- AdBlue / SCR, если есть;
- впуск;
- течи масла;
- течи антифриза;
- ошибки по всем блокам;
- дымность.

## EGR / DPF / впуск

Проверять:

- работу EGR;
- состояние DPF;
- частоту регенераций;
- давление до/после DPF;
- состояние впуска;
- следы удаления экологии;
- следы перепрошивки.

Желательно искать машину с живой экологией. Удаленный DPF / EGR без документов — красный флаг.

## Цепи ГРМ

B47 лучше старого N47 по репутации, но цепь не вечная.

Проверять:

- шум на холодном запуске;
- металлический шелест;
- ошибки по фазам;
- историю обслуживания;
- косвенные признаки большого пробега.

## Турбина

Проверять:

- свист;
- люфт;
- масло во впуске;
- ошибки по наддуву;
- тягу под нагрузкой;
- дымность.

## Коробка и xDrive

Даже с 2.0 дизелем нужно обязательно проверять трансмиссию.

Смотреть:

- ZF 8HP;
- раздатку;
- редукторы;
- кардан;
- приводы;
- одинаковость резины;
- вибрации;
- рывки при разгоне;
- ошибки по полному приводу.

Разная резина и разный износ шин — плохой знак для xDrive.

## Что не брать

Не наш вариант:

- чип;
- удаленная экология;
- нет заказ-нарядов;
- плохой холодный запуск;
- шум цепи;
- дымность;
- ошибки по топливной;
- ошибки по EGR / DPF;
- пинки АКПП;
- рывки xDrive;
- разная резина;
- мутный пробег;
- продавец не готов к диагностике.

## Идеальный вариант

Идеальный G31 520d xDrive:

- 2017–2020 год;
- B47D20;
- 190 л.с.;
- xDrive;
- без чипа;
- живая экология;
- подтвержденный пробег;
- понятная история обслуживания;
- масло двигателя каждые 7–10 тыс. км;
- обслуженная АКПП;
- обслуженная раздатка;
- одинаковая резина;
- нет ошибок;
- нет течей;
- нет шума цепи;
- цена заметно ниже 530d;
- динамика устраивает после тест-драйва.

## Итог

B47D20 / 520d xDrive — не главный, а резервный мотор для BMW 5 Touring G31.

Его можно рассматривать, если машина действительно живая и цена хорошая. Но если хочется запаса, трассовой тяги и более взрослого характера, основной выбор — 530d xDrive с B57D30.
