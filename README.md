# Car Choice Guide

Личная база по выбору возрастных премиальных автомобилей.

Цель репозитория — сохранить практичные выводы по каждой машине:

* какие моторы смотреть;
* какие моторы избегать;
* какие типовые болячки есть;
* что проверять перед покупкой;
* что делать сразу после покупки;
* какой ремонтный резерв держать.

## Машины в круге выбора
